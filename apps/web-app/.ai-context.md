# Web App - OS

This is the React frontend application for OS.

## Framework
- **React** with Vite
- **TypeScript** for type safety

## Structure
- `components/` - React components
- `features/` - Feature modules
- `pages/` - Page components
- `lib/` - Utilities

## Key Features
- Assets management
- Auth
- Automation
- Calendar
- Clients
- CRM
- Documents
- Projects
- Tracking

## Development
- Run: `npm run dev` or `pnpm dev`
- Build: `npm run build`
