# How-To Guides

**Problem-solving guides that show how to achieve a specific goal.**

How-to guides are recipes for solving specific problems. They assume you have basic familiarity with the system.

## What Makes a Good How-To Guide?

- **Goal-oriented**: Focused on achieving a specific outcome
- **Action-first**: Gets straight to the solution
- **Practical**: Assumes working knowledge
- **Flexible**: Shows multiple approaches when appropriate
