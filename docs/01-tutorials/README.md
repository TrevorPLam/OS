# Tutorials

**Learning-oriented guides that take you through a series of steps to complete a project.**

Tutorials are lessons that help you learn by doing. They're designed for newcomers who want to get started quickly.

## Available Tutorials

- [Getting Started](getting-started.md) - Set up your local development environment and run the application

## What Makes a Good Tutorial?

- **Learning-oriented**: Focused on helping beginners learn
- **Step-by-step**: Clear, numbered steps that build on each other
- **Beginner-friendly**: No assumptions about prior knowledge
- **Immediately useful**: Results in a working system
- **Repeatable**: Anyone following the steps gets the same result

## Need Help?

If you encounter issues following a tutorial, please [open an issue](https://github.com/TrevorPLam/OS/issues) with:
- Which tutorial you were following
- Which step failed
- The error message or unexpected behavior
