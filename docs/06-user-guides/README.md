# User Guides

**End-user documentation for ConsultantPro platform users.**

User guides are designed for non-technical users who need to accomplish tasks within the platform.

## Available Guides

- [Firm Admin Guide](firm-admin-guide.md) - Guide for firm administrators and owners
- [Client Portal Guide](client-portal-guide.md) - Guide for client portal users

## Audience

These guides are written for:
- Firm owners and administrators
- Firm staff members
- Client portal users
- Non-technical users

## Guide Structure

Each guide covers:
- Getting started and login
- Common tasks and workflows
- Troubleshooting common issues
- Best practices and tips
