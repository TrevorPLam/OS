## What is UBOS?

UBOS (Unified Business Operating System) is a comprehensive platform for service firms to manage:
- Client relationships and CRM
- Projects and delivery
- Billing and invoicing
- Documents and approvals
- Communications and calendar
- Automation and workflows
- And much more

## System Overview

- **Backend:** Django REST API with multi-tenant architecture
- **Frontend:** React/TypeScript application
- **Architecture:** Firm-scoped multi-tenancy with domain modules
