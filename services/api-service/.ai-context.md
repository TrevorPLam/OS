# API Service - OS

This is the Django REST Framework backend service.

## Framework
- **Django REST Framework**
- **Python**

## Architecture
Follows Clean Architecture:
- `domain/` - Business logic and domain models
- `infrastructure/` - Database, external APIs, infrastructure concerns
- `application/` - Use cases and application services
- `presentation/` - Controllers, serializers, API endpoints

## Structure
- `backend/api/` - API endpoints organized by feature
- `backend/modules/` - Domain modules with business logic
- `backend/config/` - Django settings and configuration

## Development
- Run migrations: `python manage.py migrate`
- Create superuser: `python manage.py createsuperuser`
- Run server: `python manage.py runserver`
